# AWS Resource Tracker

AWS Resource Tracker is a Bash Automation project used to collect AWS resource inventory and generate cloud governance reports.

## Services Covered

- EC2
- S3
- Lambda
- IAM
- EBS
- SES

## Features

- AWS Inventory Collection
- Unattached EBS Detection
- Stopped EC2 Detection
- S3 Report Upload
- SES Email Notification
- Logging

## Technologies

- Linux
- Bash
- AWS CLI
- Amazon S3
- Amazon SES
